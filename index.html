Port Scanner

Demo y documentación del script para detección básica de puertos.

Ver en GitHub ⤓ Descargar ZIP
Tecnologías: Python 3 • socket • uso en laboratorio
Importante: Solo usar en entornos controlados. Escanear hosts sin permiso puede ser ilegal.

Resumen

Escanea puertos comunes en un host y reporta abiertos/cerrados. Ideal para pruebas en localhost o VMs.

Cómo probar (local)

  1. Clona el repo: 
    git clone https://github.com/siralex23/portscanner.git
cd portscanner
  2. Levanta un servidor de prueba (otra terminal): 
    python3 -m http.server 8000
  3. Ejecuta: 
    python3 portscanner.py
# cuando lo pida: localhost
